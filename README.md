# OW 一键测组队工具

## 项目简介

这是一个守望先锋（Overwatch）辅助工具，用于测试对面组排情况。通过自动化操作，在选完人的等待页面一键完成：

1. **打开玩家菜单** - 长按Tab+中键呼出鼠标
2. **5组点击操作** - 右键打开菜单、图像识别join按钮点击、关闭菜单
3. **自动发送消息** - 识别小队人数并发送对应消息

## 功能特性

- ✅ **比例坐标** - 自动适配不同屏幕分辨率（推荐）
- ✅ **图像识别** - 智能识别 join.png 按钮
- ✅ **人数统计** - 识别小队图标数量（使用NMS去重）
- ✅ **自动消息** - 根据人数发送对应中文消息
- ✅ **配置文件化** - 所有参数可配置，无需修改代码
- ✅ **详细日志** - 记录所有操作步骤

## 使用方法

### 运行程序

**必须以管理员身份运行**（`keyboard` 库需要管理员权限）

右键点击 `ow_main.exe` → 选择 **"以管理员身份运行"**

### 操作流程

按 **F12** 触发一键操作：

1. **阶段一：呼出鼠标**
   - 长按 Tab → 按中键 → 松开 Tab
2. **阶段二：5组点击**
   - 右键点击玩家位置（打开菜单）
   - 图像识别并点击 join.png
   - 点击关闭菜单位置
3. **阶段三：发送消息**
   - 打开聊天框 → 截图识别小队数量 → 发送消息

## 配置文件详解 (config.json)

### 完整配置（比例坐标模式）

```json
{
    "reference_resolution": [2560, 1600],
    "coordinate_format": "值<10为比例坐标(0-1), 值>=10为绝对坐标",
    "f12_click_positions": [
        [0.281, 0.572],
        [0.281, 0.625],
        [0.281, 0.684],
        [0.281, 0.741],
        [0.281, 0.792]
    ],
    "final_click": [0.793, 0.390],
    "join_search_region": {
        "top": 0.178,
        "left": 0.129,
        "width": 0.498,
        "height": 0.637
    },
    "join_template": "join.png",
    "join_threshold": 0.8,
    "delays": {
        "tab_press": 0.025,
        "middle_click": 0.025,
        "tab_release": 0.075,
        "after_right_click": 0.125,
        "after_left_click": 0.075,
        "after_final_click": 0.075
    }
}
```

#

**推荐使用比例坐标**，自动适配任何分辨率。

### 主要配置项

| 配置项                   | 说明                   |
| --------------------- | -------------------- |
| `f12_click_positions` | 5个玩家的右键点击坐标（比例坐标）    |
| `final_click`         | 关闭菜单的点击坐标            |
| `join_search_region`  | join.png 的搜索区域（比例坐标） |
| `join_threshold`      | 识别阈值（0.0-1.0，默认0.8）  |
| `delays`              | 各操作之间的延迟（秒）          |

### 消息规则

| 识别数量 | 发送消息         |
| ---- | ------------ |
| 0    | `你跑不过我`      |
| 1    | `组排真牛，对面是两排` |
| 2    | `组排真牛，对面是二排` |
| 3    | `组排真牛，对面是三排` |
| 4    | `组排真牛，对面是四排` |
| 5    | `组排真牛，对面是五排` |
