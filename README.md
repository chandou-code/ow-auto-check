# OW测组队工具

一款基于计算机视觉和人工智能技术的守望先锋组队辅助工具，通过截图识别和自动操作简化组队测试流程。

## 项目背景

在守望先锋游戏中，组队测试时经常需要了解对面玩家数量和阵容。本工具通过屏幕截图、AI图像识别和自动化操作，帮助玩家快速完成组队测试并发送相关消息。

## 主要功能

### 智能英雄识别
- 基于深度学习模型（ResNet18）识别游戏中的英雄头像
- 支持多个英雄样本库，提高识别准确率
- 自动处理异常情况，置信度过低时提供备选方案

### 自动组队操作
- 自动执行Tab键+中键组合，打开搜索界面
- 模拟5组点击操作，自动寻找并拉人
- 支持不同分辨率的自适应坐标缩放

### 智能计数系统
- 截图分析指定区域，统计对面玩家数量
- 根据计数结果智能选择发送对应消息
- 支持发送"对面是X排"等预设消息

### 快捷消息发送
- 按F1/F2快速发送识别到的英雄相关信息
- F11发送"憋粗去"消息（带5分钟冷却机制）
- 消息内容自动复制到剪贴板，模拟Ctrl+V粘贴发送

### 截图保存
- 识别过程全程截图记录
- 保存原始图像和裁剪区域，方便调试分析

## 技术架构

### 核心技术
- **PyTorch**: 深度学习模型（ResNet18）用于特征提取
- **OpenCV**: 图像处理、模板匹配、颜色直方图分析
- **PyAutoGUI**: 自动化鼠标键盘操作
- **MSS**: 高性能屏幕截图
- **scikit-learn**: KNN分类器实现英雄识别

### 模型训练
- 使用ResNet18预训练模型提取图像特征
- KNN分类器基于余弦距离进行英雄分类
- 支持多目录样本训练，可扩展新英雄

### 坐标系统
- 采用相对坐标（0-1比例）适配不同分辨率
- 自动检测屏幕分辨率并进行坐标缩放
- 支持配置文件保存自定义坐标

## 快捷键操作

| 快捷键 | 功能说明 |
|--------|----------|
| F1 | 发送第一个识别到的英雄消息 |
| F2 | 发送第二个识别到的英雄消息 |
| F8 | 截图并进行AI英雄识别 |
| F11 | 发送"憋粗去"消息（带冷却） |
| F12 | 执行5组自动点击组队操作 |
| Ctrl+C | 退出程序 |

## 使用流程

1. **启动程序**：以管理员身份运行程序
2. **进入游戏**：在守望先锋游戏中打开组队界面
3. **识别英雄**：按F8截图，AI自动识别双方英雄
4. **发送消息**：按F1/F2发送英雄相关信息
5. **开始测试**：按F12执行自动组队操作
6. **查看结果**：程序自动计数并发送对应消息

## 注意事项

1. **管理员权限**：键盘监听需要管理员权限运行
2. **分辨率适配**：首次使用会自动适配屏幕分辨率
3. **坐标学习**：删除config2.json可重新学习坐标位置
4. **冷却机制**：F11功能有5分钟冷却时间

## 配置文件说明

### config.json
主配置文件，包含各操作的延迟时间、点击坐标、模板路径等参数。

### config2.json
相对坐标配置文件，由程序自动生成，记录学习到的坐标位置。
