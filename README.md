# OW 一键测组队

用于《守望先锋》自动测试对面组排情况的脚本。

## 功能说明

在选完人的等待页面，按下 F12 即可自动执行以下操作：

1. **呼出小队界面** - 按住 Tab + 中键打开小队列表
2. **自动点击队友位置** - 按预设坐标依次点击各位置，拉取对面玩家信息
3. **识别组队人数** - 截取左下角聊天区域，通过图像识别统计对面在小队中的个数
4. **自动发送消息** - 根据识别结果自动在聊天框发送：
   - 0人组队：`你跑不过我`
   - 1人组队：`组排真牛，对面是两排`
   - 2-5人组队：`组排真牛，对面是X排`

## 使用方法

### 环境要求

- Python 3.7+
- Windows 操作系统

### 安装依赖

```bash
pip install opencv-python numpy mss pyautogui pyperclip keyboard
```

### 运行

```bash
python ow_main_ratio.py
```

### 按键说明

| 按键 | 功能 |
|------|------|
| F12 | 执行完整一键测组队流程 |
| Ctrl+C | 退出程序 |

## 配置说明

### 坐标配置

所有坐标均为**比例值**（0.0-1.0），自动适配不同分辨率屏幕。

```python
# F12 点击位置（比例）
F12_CLICK_RATIOS = [
    (0.280859375, 0.571875),  # 第1组
    (0.280859375, 0.625),     # 第2组
    # ...
]

# 检测区域（比例）
DETECT_RATIO = {
    "left": 0.0234375,        # 60 / 屏幕宽度
    "top": 0.695,             # 1112 / 屏幕高度
    "width": 0.2265625,       # 580 / 屏幕宽度
    "height": 0.23,           # 368 / 屏幕高度
}
```

### 消息配置

```python
# 识别到0人时发送
MESSAGE_ZERO = "你跑不过我"

# 识别到1人时发送（1视为2）
MESSAGE_ONE = "组排真牛，对面是两排"

# 识别到2-5人时发送
MESSAGE_OTHER = "组排真牛，对面是{}排"
```

### 模板图片

将待识别的图标放入 `picture` 目录：

- `b7dcf52163eb26cf4c1a157c40f9ca48.png` - 用于识别小队中的玩家图标
- `img.png` - 可选，用于检测特定图片出现时自动发送消息

## 工作原理

1. **模板匹配** - 使用 OpenCV 的 `matchTemplate` 函数进行图像识别
2. **非极大值抑制(NMS)** - 过滤重叠的匹配结果，确保计数准确
3. **比例换算** - 所有坐标使用相对比例，适配不同分辨率

## 注意事项

- 首次使用建议先手动测试各坐标位置是否正确
- 确保 `picture` 目录下的模板图片清晰可识别
- 如识别不准确，可调整 `COUNT_THRESHOLD` 阈值（默认0.9）
